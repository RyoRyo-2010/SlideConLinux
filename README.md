# Slide Con Linux
PowerPointなど，スライドをリモコンから操作するためのシステム「Slide Con」の受信プログラムです．

Linux向けに，完全にネイティブなCコードで書かれています．疲れた．

## 実行の流れ
シリアルポートから`left\n`か`right\n`を受け取ると，右矢印キー，左矢印キーを代わりに押してくれます．これだけですがプログラミングすごい大変でした．

## 実行方法
まず，コンパイルします．
```
gcc main.c -o slidecon
```
次に，実行します．
```
sudo ./slidecon /dev/ttyACM0
```
おそらく殆どの環境ではroot権限が必要でしょう．

## ライセンス
こんなプログラムでOSSを名乗ってすみません．ただ少しでも自分や他の方にシリアルポート周り，User Input周りの参考にして貰えたらと．
